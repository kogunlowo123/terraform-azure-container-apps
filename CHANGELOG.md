# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added

- Container Apps Environment with configurable workload profiles
- Container App deployments with multi-container support
- Ingress configuration with traffic splitting and IP restrictions
- Dapr component integration (state stores, pub/sub)
- Custom domain and managed certificate support
- Log Analytics workspace integration (create or bring-your-own)
- VNet injection with internal load balancer option
- Zone redundancy support
- Azure File Share storage mounts
- Health probes (liveness and readiness)
- Container registry authentication
- Secret management
- Basic, advanced, and complete usage examples
